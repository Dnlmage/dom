
Описание
Тестово задание реализовано в виде API имеющего 2 матода, POST и GET

Установка

1) Склонировать проект
2) Для запуска надо выполнить скрит dump.sql
3) Выполнить команду composer install
4) Настроить хост на файл index.php в корне
5) Данные настройки конекта к БД лежат в файлу DbConnection.php

Для проверки можно использовать консольную утилиту httpie
Примеры запроса
1) http -f POST http://engldom message="messs:)"
2) http -f GET http://engldom
